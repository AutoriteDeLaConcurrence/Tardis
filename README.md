# TARDIS

This project aims at creating a database of public full-text decisions of competition authorities members of the ICN (International Competition Netowrk) in a machine-readable format.

This involves defining a data format (e.g., JSON or Parquet), extracting text from published decisions (often PDFs), and hosting the resulting data on a shared platform. 

Such a resource could improve analysts’ ability to locate decisions related to specific firms across multiple jurisdictions and enable AI-powered tools.

## Data

A l'heure actuelle, contient les données des autorités suivantes :

- Autorité de la concurrence (France)
- Autorité de la concurrence de nouvelle calédonie (New Caledonia)
- Competition COmmission (Switzerland)
